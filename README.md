## Instructions:

__Select list in this format:__

    username|password|*

__* is a wildcard for any additional data.__

__Example:__

    acc1|pw1|
    acc2|pw2|Some Country
    acc3|pw3|Korea|LEVEL100
    accN|pwN|rnd!"§%2/-äÜ!"§*'!

__Happy checking!__
