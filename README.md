LoLChecker
==========


![LoLChecker IMAGE](http://abload.de/img/unbenannt22rg7.png "LoLChecker")


## Instructions

__Select list in this format:__

    username|password|*

___* is a wildcard for any additional data.___

__Example:__

    acc1|pw1|
    acc2|pw2|Some Country
    acc3|pw3|Korea|LEVEL100
    accN|pwN|rnd!"§%2/-äÜ!"§*'!

__Happy checking!__
